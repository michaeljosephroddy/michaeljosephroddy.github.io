Hey there, hows it going? my names Michael, a technology enthusiast, gym goer and general all round curious thinker based in Ireland.

I wanted to start this blog to express my thoughts on all things that interest me, giving my expert, arm chair detective opinion on topics ranging from programming concepts and software development to religion and health and fitness.

So don't be surprised if one week you see a blog post demonstrating how to use the Stream API in Java or devling into the deeply spiritual and theological purposes of the death and ressurection of Jesus Christ, and the following week discussing how using intensity techniques in your workouts are a great way to go beyond failure.

I want this blog to be a place where I can discuss a diverse range of topics, expressing my personal thouhgts, feelings and opionions without conforming to a single subject area. If you think that content sounds like you, then stick around! I'll be aiming to do at least one blog post per week.

Thanks for taking the time to checkout my blog, hope you have a good day. Mike.
